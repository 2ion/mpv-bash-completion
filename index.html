mpv-bash-completion

Bash completion for the mpv video player

View on GitHub Download .zip Download .tar.gz

A script for generating a Bash completion script for the popular mpv video player. It features...

  • Completion for all --options,
  • Type-based completion for --option arguments for choices, flags, integers and floats,
  • Completion for upper/lower bounds for integer- and float-type argument ranges where applicable,
  • Regular file name completion.

Installation

Besides using the methods below, you can always generate the script manually. See the README for instructions.

The completion will work on any system running GNU Bash 4. Mac OSX users will have to install Bash 4. The default Bash 3 OSX is shipping lacks a required shell built-in.

The script can be built on any system having, besides mpv, the following programs available:

  • GNU Bash 4 or greater
  • GNU coreutils, or BSD alternatives for grep, sed, cut, column, tail

Arch Linux

mpv-bash-completion-git is available in the Arch User Repository and can be built using the ABS. See here for the PKGBUILD.

Debian

You can build a package locally by following the instructions on the debian branch.

Pre-built packages are available for download here, though they may be obsolete.
